В данной папке представлены синтетические данные [DiffusionAE](https://arxiv.org/abs/2311.01452). Источник загрузки доступен по [ссылке](https://github.com/fbrad/DiffusionAE), а также сами файлы представлены в репозитории.

Данные представлены в виде набора следующих файлов:
- `labels.npy` -- тестовая разметка
- `labels_validation.npy` -- валидационная разметка (model selection)
- `test.npy` -- тестовые временные ряды (**с разметкой**)
- `train.npy` -- обучающие временные ряды (**без разметки**)
- `validation.npy` -- валидационные данные (**с разметкой**)

Синтезировано по 50к наблюдений в каждом. Размерность везде 5, но аномалии содержит лишь одно измерение. Выборки стратифицированы по доле анормальности.
